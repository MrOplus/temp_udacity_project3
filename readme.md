# NLP News AI Evaluation
## How to run Project : 
copy .env.example to .env and fill the required fields
### Run Development Server
```bash
npm run build-dev
npm run server-dev
```

### Run Production Server
```bash
npm run build-prod
npm run server-prod
```

### Run Test
```bash
npm run test
```

### Start Production Server
```bash
npm start start-prod
```
